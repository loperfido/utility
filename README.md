# 🛠️ Utility Hub

Una collezione di tool online gratis per dev e professionisti. Tutto fatto in PHP con un'interfaccia semplice e intuitiva.

## ✨ Cosa c'è dentro

- **34 strumenti** diversi
- Interfaccia moderna e responsive
- Ricerca e ordinamento integrati
- Zero registrazione
- Totalmente gratuito

## 🧰 Gli Strumenti

### 🔢 Convertitori
- **Convertitore Basi** - Passa da una base numerica all'altra
- **Convertitore Colori** - RGB, HEX, HSL e tutto il resto
- **Convertitore Unità** - Converte un po' di tutto
- **Convertitore Pixel** - Pixel/em/rem, no problem
- **Convertitore Timestamp** - Unix timestamp ↔ data
- **Case Converter** - Trasforma il testo (upper, lower, title...)

### 📅 Date e Tempo
- **Differenza Date** - Quanti giorni tra due date?
- **Aggiungi Giorni** - Aggiungi/togli giorni da una data
- **Timezone Converter** - Fusi orari, finally capito
- **Giorni Mancanti** - Countdown per quella data importante

### 🔐 Sicurezza e Crittografia
- **Generatore Password** - Password a prova di hacker
- **Password Strength** - La tua password fa schifo? Te lo dice qui
- **Generatore UUID** - UUID unici al volo
- **Hash Generator** - MD5, SHA1, SHA256...
- **AES Encrypt/Decrypt** - Cifra e decifra con AES
- **Cifrario di Cesare** - Come gli antichi romani
- **Checksum File** - Calcola il checksum dei tuoi file

### 🇮🇹 Roba Italiana
- **Codice Fiscale** - Calcola e verifica il CF
- **Generatore IBAN** - IBAN validi in un attimo

### 🧮 Calcolatrici
- **Calcolatore BMI** - Sei in forma?
- **Calcolo Sconto** - Quanto ti sconto davvero?
- **Interesse Composto** - Roba da finanziari
- **Calcolatore Mutuo** - Simula il tuo mutuo

### 📝 Testo e Stringhe
- **Contatore Parole** - Conta parole e caratteri
- **Inverti Testo** - Metti tutto al contrario
- **Estrai Email** - Tira fuori le email da un testo
- **Lorem Ipsum** - Testo finto per i tuoi mockup

### 🎲 Generatori
- **Generatore QR Code** - QR code al volo
- **Generatore IP** - Genera indirizzi IP
- **Numeri Casuali** - Numeri random, perché no
- **WhatsApp Link** - Crea link per WhatsApp

### 🌈 Altro
- **Contrasto Colori** - I tuoi colori sono accessibili?
- **Base64** - Codifica/decodifica Base64
- **Calcolatore Mutuo** - Simula prestiti (sì, è duplicato 😅)

## 🚀 Come si usa

### Cosa ti serve
- PHP 7.4 o più nuovo
- Un server (Apache, Nginx, quello che vuoi)

### In locale

1. Clona il repo:
```bash
git clone https://github.com/tuo-username/utility-hub.git
cd utility-hub
```

2. Lancia il server PHP:
```bash
php -S localhost:8000
```

3. Apri `http://localhost:8000` e sei dentro

### Online

Butta tutti i file nella root del tuo server (tipo `/var/www/html/`) e via.
